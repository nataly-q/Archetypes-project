# Archetypes-project
Концепт теста на архетипы с визуализацией и подсчётом баллов.

Логика теста: 
- пользователю предлагается 8 вопросов;
- в каждом 3 варианта ответа;
- в итоге - 1 архетип + 3 "дополнительных".
Всего 12 архетипов.

Описание архетипов учитывает "личные качества" + использование данных архетипов в рекламных кампаниях.
