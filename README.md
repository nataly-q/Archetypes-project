# Archetypes-project
Концеп: тест на архетипы с визуализацией и подсчётом баллов.

Логика теста: 
- пользователю предлагается 8 вопросов;
- в каждом 3 варианта ответа;
- в итоге - 1 ведущий архетип + 2 "дополняющих".
Всего 12 архетипов.

В описании архетипов представлены не только "личные качества", но и связь с рекламными кампаниями, которые используют данный архетип. 

Распределение дополнительных идет в процентах.

- работа с JSON;
